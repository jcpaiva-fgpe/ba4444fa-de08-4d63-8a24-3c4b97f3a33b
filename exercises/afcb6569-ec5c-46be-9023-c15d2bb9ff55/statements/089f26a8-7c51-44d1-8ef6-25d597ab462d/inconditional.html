Para manipular los datos existentes en una base de datos relacional, vamos a utilizar el lenguaje SQL. En concreto, si lo que queremos es extraer un subconjunto de los datos existente, utilizaremos el comando SELECT.

Lo primero que vamos a hacer es mostrar todos los datos de una determinada tabla, sin ninguna restricción.

Ejemplo:

si queremos mostrar todos los datos de la tabla regions haremos:

SELECT * FROM regions

Lo único que debemos hacer es escribir el nombre de la tabla (regions) detrás del texto de SQL SELECT * FROM, obteniendo la siguiente tabla:

REGION_ID REGION_NAME
1 Europe
2 Americas
3 Asia
4 Middle East and Africa
